# CounselGPT
한국어 심리 상담 데이터셋   
--------------------    
본 데이터셋은 질의, 답변 전량 openai chatgpt api를 통해 구축되었습니다.  

* total_kor_counsel_bot.jsonl    
  * 총 13,234 싱글턴 장문 데이터
  * input: 고민 내용    
  * output: 상담사 답변
 
* total_kor_multiturn_counsel_bot.jsonl
  * 총 8,731 멀티턴 대화 데이터
  * speaker: 발화자
  * utterance: 발화 내용
